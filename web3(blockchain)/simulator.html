💡 How Mining Works

  1. 1 Mining searches for a nonce that produces a valid hash when combined with the block data and previous hash.
  2. 2 A valid hash begins with "00" — this is the proof-of-work difficulty target.
  3. 3 Changing any data in a block completely changes its hash due to the avalanche effect of SHA-256.
  4. 4 Since each block stores the previous block's hash, altering one block breaks the chain — all subsequent blocks become invalid.
  5. 5 To restore the chain, every block after the tampered one must be re-mined. This is why blockchain is tamper-resistant.

Current Difficulty: Hash must start with "00" (2 leading zeros). Real Bitcoin requires ~72 leading zeros, making it computationally exponentially harder.
