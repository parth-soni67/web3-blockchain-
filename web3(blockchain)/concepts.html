Core Web3 Concepts

Learn the building blocks of Web3 through clear visual comparisons. Each card contrasts old and new paradigms side by side.

🌐

Web2 vs Web3

The internet evolution

🔴 Web2
Centralized platforms
Companies own data
Requires intermediaries
Limited transparency
🟢 Web3
Decentralized networks
Users control data
Peer-to-peer interactions
Transparent blockchain records

Web3 represents a fundamental shift from corporate-controlled platforms to user-owned, decentralized ecosystems powered by blockchain technology.

₿

Ethereum vs Bitcoin

Two blockchain giants

₿ Bitcoin
Digital currency
Payments
Fixed functionality
Store of value
Ξ Ethereum
Blockchain platform
Smart contracts
Programmable
Decentralized apps (dApps)

Bitcoin pioneered digital scarcity and peer-to-peer money. Ethereum extended blockchain with Turing-complete smart contracts, enabling an entire ecosystem of decentralized applications.

🔑

Public Key vs Private Key

Cryptographic key pairs

🔓 Public Key
Shared openly
Receive crypto
Wallet address
Safe to share
🔒 Private Key
Secret
Sign transactions
Ownership proof
Never share!

In Web3, your private key is your identity. Anyone with your private key controls your assets. Always store it securely — there is no password reset.

🗄️

Blockchain vs Traditional Database

Data storage paradigms

🔴 Database
Centralized
Editable / deletable
Controlled by admins
Limited transparency
🟢 Blockchain
Decentralized
Immutable
Cryptographically secured
High transparency

Traditional databases offer speed and flexibility but require trust in a central authority. Blockchain sacrifices some performance for trustlessness, immutability, and openness.

Key Terminology

⛏️

Mining

The process of finding a nonce that produces a hash with a required prefix, validating a new block.

#️⃣

Hash Function

A one-way function that maps arbitrary data to a fixed-length digest. Any change in input completely changes the output.

📜

Smart Contract

Self-executing code stored on the blockchain. Runs automatically when predefined conditions are met.

🎨

NFT

Non-Fungible Token. A unique digital asset on a blockchain, proving ownership of digital content.

🔄

DeFi

Decentralized Finance. Financial services built on blockchain without traditional banks or intermediaries.

🔐

Wallet

Software that stores private/public key pairs and allows users to interact with blockchain networks.
